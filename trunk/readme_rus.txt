Проект игрового "движка" RGDE

Описание структуры директорий проекта:
\
  -external - внешние библитеотеки которые используются для сборки проекта (boost, tinyXML, ogg, vorbis)

  -lib      - сюда после компиляции проекта будут помещены скомпиленные библиотеки
  -examples - директория с обучающими проектами по работе с движком

  -rgdengine\ 
     -src   - место где искать исходный код движка :)  
     -rgde  - здесь находятся все h файлы проекта, необходимые для использования движка

  -temp     - мусор от компиляции

  -tols\     
      - bin - откомпиленные версии утилит
      - src - исходные коды утилит

  readme.txt - файл описания который вы сейчас читаете :)


Для того что бы начать использовать движок, необходимо прописать в настройках своего IDE(или проекта) полные пути к директориям lib и rgdengine. А так же к boost 1-34 и gmtl.
