Проект игрового "движка" RGDE

Описание структуры директорий проекта:
\
  -external - внешние библитеотеки которые используются для сборки проекта (boost, tinyXML, ogg, vorbis)
  -include  - здесь находятся все h файлы проекта, необходимые для использования движка
  -lib      - сюда после компиляции проекта будут помещены скомпиленные библиотеки
  -samples  - директория с обучающими проектами по работе с движком
  -src      - место где искать исходные код движка :)
  -temp     - мусор от компиляции
  -tols     
      - bin - откомпиленные версии утилит
      - src - исходные коды утилит
  readme.txt - файл описания который вы сейчас читаете :)


Для того что бы начать использовать движока, необходимо прописать в настройка своего IDE полные пути к директориям lib и include.
