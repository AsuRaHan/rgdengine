Game Engine Project: RGDE

Discription of directory structure
\
  -external - external libs used in engine build
  -include  - all public headers are here 
  -lib      - build target directory
  -samples  - directiry with sample projects
  -src      - engine source code
  -temp     - build garbage
  -tols     
      - bin - build output for tools
      - src - tools source code
